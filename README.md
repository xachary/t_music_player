# T Music Player

这是一款完全离线的音乐播放器安卓App，适用于追求纯粹听歌体验的朋友们。如果你有自己的音乐文件库，本应用也许适合你。

[下载地址](https://github.com/xachary/t_music_player/releases)

## 其它小细节

- 播放的时候持续亮屏的
- 快速定位当前播放歌曲的列表位置
- 交互有震动
- 列表中正在播放的歌曲高亮会微微闪烁
- 播放失败，大概就是音频文件有损坏，会显示并通知提醒你
- 隐藏状态栏
